## Installation Instructions

1. Clone the repository
   ```bash
   git clone https://github.com/Girish2513/Manoj-Brothers-Extension.git
   cd Manoj-Brothers-Extension
   ```
2. Install dependencies
   ```bash
   npm install
   ```

## Usage

To use the extension, open your browser and navigate to [manojbrothersextension.com](http://manojbrothersextension.com)
